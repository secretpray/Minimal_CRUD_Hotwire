# Minimal_CRUD_Hotwire

<details>
  <summary>Getting Started</summary>

  ## Check:

  * ruby -v (3.1.0)

  * rails -v (7.1.0)

  ## Install:

  * git clone git@github.com:secretpray/Minimal_CRUD_Hotwire.git

  * cd Minimal_CRUD_Hotwire

  * bundle

  * bin/rails db:setup
  
  * yarn install --check-files
  
  * yarn run build
  
  * yarn run build:css
  
  * redis-server --daemonize yes
  
  * bin/dev

</details>
